# WebGPU game of life

An entirely GPU-based game of life implementation, running at a smooth 60FPS on the web.

> Based on [Google's WebGPU codelab](https://codelabs.developers.google.com/your-first-webgpu-app).

<img src="preview.png" />

## Usage

```sh
npm start
```
